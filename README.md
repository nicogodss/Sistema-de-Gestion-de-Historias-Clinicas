# Patient Management System (C++)

## Description
Console-based system developed in C++ for managing patient medical records. The system allows registering patients, storing consultations, and retrieving clinical history efficiently.

This project was built as part of an academic assignment focused on structured programming and data management.

## Problem Context
The system simulates a healthcare institution that requires an organized way to manage patient information, avoiding data loss, duplication, and inefficient access to medical records :contentReference[oaicite:0]{index=0}.

## Features
- Register new patients with personal data
- Store medical consultations associated with each patient
- View complete clinical history of a patient
- Modify existing diagnoses
- Logical deletion of consultations
- Load data from binary files
- Generate reports in text format

## Technical Implementation
- Dynamic linked lists using templates
- Handling of character arrays (char[])
- Binary file storage for persistence
- Text file generation for reports
- Structured programming approach

## Data Structures
- Patients stored in linked lists
- Each patient contains a list of medical consultations
- Use of structs for data modeling

## Technologies
- C++
- File handling (binary and text)
- Pointers and dynamic memory

## Purpose
Academic project focused on applying:
- Data structures (linked lists)
- File management
- Structured programming
- Data persistence
